- Antes de ler a biblia analisar o contexto.

- 5 métodos antes da leitura bibíica {
	- Perguntas arqueológicas.
		- quem escreveu?
		- Quando escreveu? (Quanto tempo escreveu depois de Jesus).
		- Foco para quem foi escrito?
		- Qual o estilo literário do escritor?
		- Por que escreveu o livro?
}

- estudo biblico induitivo (EBI) {
	- Observação; {
		- O que o texto diz?
		- Dicas {
			- Imprimir o Texto;
			- Ler várias vezes;
			- Anotações para organizar o pensamento. Codificar o texto!!!! (Separar por pessaos, tempo, promessas, etc...)
		- Dicionário;
		- Ler outras traduções;
		- Fazer resumo/ esboço.	
		}
}
	- Interpretação; {
		- Um versículo nunca vai significar para mim o que não significou para a audiência original.
		- Referências cruzadas;
		- Paráfrase;
		- Bíblia de estudo/ comentários bóblicos.
		}
	- Aplicação;{
		- Como isso deveria me mudar /moldar;{
			- O que o autor quis dizer no original que serve pra nós (verdades eternas.
		}		
		- Responder questionamentos:{
			- O que o texto me ensina sobre Deus?
		- A luz do que aprendi sobre Deus, o que ele ensina sobre mim?
		- Como responderei a isso?
		}
		}
}
